# Service Metrics (Monitoring, Auditing, BI)

Suggested metrics aligned to TXDPS services:

- **Uptime % (SLA/SLI)** – target by business criticality.
- **MTTD/MTTR** – Mean Time To Detect/Resolve incidents.
- **Change Success Rate** – percentage of successful changes without incidents.
- **Deployment Frequency** – per environment per time window.
- **Lead Time for Changes** – from commit to production.
- **Security** – Vulnerabilities open by severity, Patch latency, MFA coverage.
- **Capacity** – CPU/Memory storage headroom, queue depth.
- **Support** – First response time, backlog age.
- **User Lifecycle** – Provision time for onboarding, Deprovision SLA for offboarding.
